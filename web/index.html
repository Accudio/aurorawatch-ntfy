AuroraWatch UK ntfy notification bridge

An unofficial and simple way of getting AuroraWatch UK notifications on devices via ntfy.

Uses the AuroraWatch UK API to check aurora status and ntfy for delivering notifications.

How to use this?

First, install the ntfy app on your chosen device:

  • Google Play Store (Android)
  • Apple app store (iOS)
  • Progressive Web App (Windows, macOS, linux, other)
  • F-Droid (Android)

Then, subscribe to the aurorawatch topic on the default ntfy.sh server.

For more information about AuroraWatch UK, their API, and Auroras in general visit the AuroraWatch UK website. This website and project was made by Alistair Shepherd.
